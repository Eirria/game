# 🛡️ Life Balance System — Roadmap

Система жизненного баланса: веб-приложение для отслеживания ресурсов в четырёх сферах жизни.
Метафора сосудов и переливов. События (баффы, дебаффы, ивенты) перераспределяют ресурсы между сосудами.

---

## MVP 🚧 (в разработке)

**Срок:** 2 недели
**Суть:** один сосуд, один тип события. Работающий продукт, можно показать.

- Один сосуд: Физическое здоровье (прогресс-бар, текущее/максимальное значение)
- Один тип события: Баффы (карточки с названием, прогрессом, статусом)
- Кнопка «Обновить» — активирует бафф и добавляет ресурс в сосуд
- Статусы: active / completed / idle (зависят от прогресса)
- Сохранение в localStorage
- Форма добавления нового баффа
- Адаптивная вёрстка
- Деплой на Vercel

---

## v1.0 📋 (спроектирована)

**Срок:** +4 недели после MVP
**Суть:** полноценный дашборд с переливами.

- 4 сосуда: Физическое здоровье, Ментальное состояние, Повседневная жизнь, Финансы
- Дашборд с визуализацией всех сосудов
- Типы событий:
  - Баффы — добавляют ресурс
  - Дебаффы — отнимают ресурс
  - ТО — обязательная забота (несгораемый минимум)
  - Ивенты — разовые события (могут добавлять и отнимать)
- Механика перелива: событие тратит из одних сосудов, добавляет в другие
- Модель delta[] — матрица изменений по сосудам
- История событий с фильтрацией по типу и дате
- Кастомные хуки (useVessels, useEvents, useLocalStorage)
- Сохранение и миграция данных MVP → v1.0
- ARCHITECTURE.md с описанием модели данных и паттернов

---

## v2.0 💡 (концепция)

**Срок:** без жёсткого дедлайна
**Суть:** геймификация, облачное хранение, мобильная версия.

- Квесты — задачи с наградой при выполнении
- Регулярные события — автосрабатывание по расписанию
- Геймификация: уровни, достижения, визуальные награды
- Облачное хранение: Firebase / Supabase
- Авторизация: Google OAuth
- PWA: офлайн-режим, установка на телефон
- TypeScript — миграция с JavaScript
- Тесты: Vitest + React Testing Library
- CI/CD: GitHub Actions
